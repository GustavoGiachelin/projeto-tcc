Objetivo

Neste desafio, você percorrerá 3 labirintos gerados por diferentes algoritmos. Seu objetivo é encontrar o troféu presente no final de cada um dos labirintos. Você será teleportado para o proximo labirinto quando encontrar o troféu.


Instruções

- Use as teclas W, A, S e D para se movimentar.

- Use o mouse para movimentar a câmera.


- Clique no botão "PLAY" quando estiver pronto para começar.



Que a sorte esteja sempre ao seu favor!

Goal

In this challange you will run through 3 mazes generated by different algorithms. Your goal is to find the trophy in the end of each one of the mazes. You will be teleported to the next maze once you find the trophy.


Instructions

- Use the mouse to move the camera.


- Hit the "PLAY" button when you're ready to go.
